Estilo Externo

O CSS externo é uma maneira de separar o conteúdo (HTML) da estilização (CSS), deixando o código mais organizado e reutilizável. Ele é aplicado a uma página HTML através de um arquivo .css separado, que é referenciado no cabeçalho do documento HTML.


  1. HTML (index.html):
    • A estrutura da página HTML contém o conteúdo que será estilizado.
    • O arquivo CSS externo (estilo.css) é referenciado no cabeçalho do HTML com a tag <link rel="stylesheet" href="estilo.css">.
  2. Css (estilo.css):
    • O arquivo CSS externo contém todas as regras de estilização, separadas do conteúdo HTML.
    • Ele define as cores dos títulos (h1 e h2), a cor de fundo da página, e a justificação dos parágrafos (p)
