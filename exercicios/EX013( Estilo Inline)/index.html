Estilos Inline

  • Estilos inline são uma maneira de aplicar CSS diretamente em um elemento HTML usando o atributo style.
    Diferente das folhas de estilo externas ou embutidas, os estilos inline afetam apenas o elemento no qual estão definidos,
    tornando-os úteis para ajustes rápidos e específicos.
